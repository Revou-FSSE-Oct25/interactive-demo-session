## Challenge 1: SQL Code Challenges

Solve the following SQL challenges from HackerRank:

### 1. Employee Salaries

Write a query that prints a list of employee names for employees in the Employee table having a salary greater than a specific amount per month who have been employees for less than a specific number of months. Sort your result by ascending employee_id.

[HackerRank Challenge Link](https://www.hackerrank.com/challenges/salary-of-employees/problem?isFullScreen=true)

### 2. Top Earners

Write a query to find the maximum total earnings for all employees as well as the total number of employees who have maximum total earnings.

[HackerRank Challenge Link](https://www.hackerrank.com/challenges/earnings-of-employees/problem?isFullScreen=true)

### 3. The Blunder

Calculate the difference between the actual average salary and the miscalculated average (where the 0 key was broken during calculation).

[HackerRank Challenge Link](https://www.hackerrank.com/challenges/the-blunder/problem?isFullScreen=true)

### 4. Population Census

Query the sum of the populations of all cities where the CONTINENT is 'Asia'.

[HackerRank Challenge Link](https://www.hackerrank.com/challenges/asian-population/problem?isFullScreen=true)

### 5. Top Competitors

Write a query to print the hacker_id and name of hackers who achieved full scores for more than one challenge, ordered by the total number of challenges in which they earned a full score.

[HackerRank Challenge Link](https://www.hackerrank.com/challenges/full-score/problem?isFullScreen=true)
