# Week 22 Challenge - Prisma Relations & JWT Authentication

## Challenge 1: Implementing Relations in Prisma

The first challenge involves working on schema.prisma file to implement various types of relationships between entities:

- One-to-Many relationship
- Many-to-Many relationship
- Many-to-One relationship
- One-to-One relationship
- Implementing service methods to demonstrate these relationships

## Challenge 2: JWT Authentication in NestJS

The second challenge focuses on auth.service.ts implementing a secure authentication system using JWT:

- User registration and login
- Protected routes with Guards
- Custom decorators for accessing the current user
- Secure password handling
